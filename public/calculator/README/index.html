Simple Calculator Exercise

This task manager was created (with zero assistance) based on the exercise article: https://jscrambler.com/blog/html-css-and-javascript-practice-projects-to-level-up-your-developer-skills

Features

  • Prioritization of multiplication and division
  • All operations and numbers stored in array
  • Pressing operation post operation will use the previous answer to calculate next operation.
